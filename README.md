# Calculator

A simple calculator application built with React.

## Features

- Addition, subtraction, multiplication, and division
- Reset input field
- Reset displayed result
- Reset all functionality
- Error handling for division by zero

## Tech Stack

- React
- Vite
- CSS

## Getting Started

```bash
npm install
npm run dev
```

## Learning Objectives

- Manage application state using useState
- Access input values using useRef
- Implement arithmetic operations
- Handle edge cases (division by zero)
- Connect UI elements with event handlers
